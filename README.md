# GOT-main

A cinematic Game of Thrones-inspired website built with React and Vite. It uses a full-screen HD video hero, scroll-based chapter transitions, and interactive house cards.

## Languages Used

- JavaScript / JSX
- CSS
- HTML
- Node.js tooling

## Tech Stack

- React
- Vite
- GSAP / ScrollTrigger
- ESLint

## Open Directly

You can open the static version without running a server:

```text
E:\GOT-main\index.html
```

## Run Locally

```bat
cd /d E:\GOT-main\vite-project
npm.cmd install
npm.cmd run dev
```

Local URL:

```text
http://127.0.0.1:5173
```

## Build And Publish Static Files

```bat
cd /d E:\GOT-main\vite-project
npm.cmd run publish:root
```

This rebuilds the Vite app and updates the direct-open files in the root folder.
